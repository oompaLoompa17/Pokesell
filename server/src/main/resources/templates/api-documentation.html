Pokémon TCG API Documentation

Learn how to use the Pokémon TCG API to fetch card data and perform other operations.

Overview

The Pokémon TCG API provides endpoints to fetch Pokémon card details, manage user favorites, and perform searches.

API Endpoints

Get All Cards

Endpoint: /api/raw

Method: GET

Description: Returns a list of all Pokémon cards in raw JSON format.

Hyperlink: Click on this to view the entire JSON data

Response Example:

{
    "id": "xy7-54",
    "name": "Pikachu",
    "types": ["Electric"],
    "rarity": "Common",
    "set": {
        "id": "xy7",
        "name": "Ancient Origins"
    }
}

Search Cards

Endpoint: /cards/search

Method: GET

Description: Search for Pokémon cards using filters like name, type, set, or rarity.

Parameters:

  • name (optional) - Card name (e.g., "Pikachu").
  • type (optional) - Pokémon type (e.g., "Electric").
  • set (optional) - Set ID.
  • rarity (optional) - Rarity of the card (e.g., "Common").
